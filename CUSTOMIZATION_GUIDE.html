How to Change Colors

1

Open styles.css

Navigate to the :root section at the top of the file.

2

Find the color variable

:root {
  --primary-blue: #6366f1;
  --secondary-blue: #4f46e5;
  /* ... */
}
3

Replace with your hex code

--primary-blue: #YOUR_COLOR_HERE;
4

Save and refresh

All elements using that variable will update automatically.

Gradient Customization

Many elements use gradient backgrounds. Search for linear-gradient in styles.css: 

/* Primary button gradient */
background: linear-gradient(
  135deg,
  var(--primary-blue) 0%,
  var(--secondary-blue) 100%
);

/* You can modify:
   - Angle: 135deg ‚Üí 90deg, 180deg, etc.
   - Colors: Any hex or CSS variable
   - Stops: 0%, 50%, 100% positions */
üí° Color Tool: Use Coolors.co to generate harmonious color palettes, or UI Gradients for gradient inspiration.

Modifying Animation Speed

Find the animation in styles.css: 

.hero-badge {
  animation: fadeInUp 0.8s cubic-bezier(0.34, 1.56, 0.64, 1) 0.1s both;
  /*         ^name    ^dur  ^easing                            ^delay  ^mode */
}

Speed Adjustments:

  • Faster: Change 0.8s ‚Üí 0.4s or 0.5s
  • Slower: Change 0.8s ‚Üí 1.2s or 1.5s
  • Much slower: Change to 2s for dramatic effect

Disabling Animations

To disable all animations (good for accessibility): 

/* Add to styles.css */
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}

Creating Custom Animations

1

Define keyframes

@keyframes slideInLeft {
  from {
    opacity: 0;
    transform: translateX(-50px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}
2

Apply to element

.my-element {
  animation: slideInLeft 0.8s ease-out forwards;
}

Intersection Observer

Cards animate when scrolled into view:

script.js - Lines 194-219 
const observer = new IntersectionObserver(function (entries) {
  entries.forEach((entry) => {
    if (entry.isIntersecting) {
      entry.target.style.animation = "fadeInUp 0.6s ease forwards";
      observer.unobserve(entry.target);
    }
  });
}, observerOptions);
üí° Performance: Use transform and opacity for animations. They're GPU-accelerated and won't cause reflows.
